Exercício 12

Criar uma função palavraAEncontrar, que devolve o número de ocorrências de uma letra (passada como parâmetro) numa frase.

Por exemplo:

                    
                        function palavraAEncontrar() {
                            ...
                        }
                        let frase="O filho foi visitar o pai.";
                        palavraAEncontrar("o");

O resultado será 3, que é o número de vezes que a letra "o" ocorre na frase (a maiúscula não entra para este cálculo). Podem utilizar métodos de strings.

Variante

Pretende-se que a função devolva o número de ocorrências de uma letra (passada como parâmetro) numa frase, independentemente de estar em maiúsculas ou minúsculas.

Solução

                    
                        function palavraAEncontrar(letter) {
                            let letterCount = 0;
            
                            frase.split("").forEach((i) => {
                                letterCount += i == letter ? 1 : 0;
                            });
            
                            return letterCount;
                        }
            
                        let frase = "O filho foi visitar o pai.";
                        console.log(palavraAEncontrar("o"));
