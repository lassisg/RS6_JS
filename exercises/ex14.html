Exercício 14

Dado um array de valores inteiros,


                    [1,2,4,5,10,6,3]

Criar um array que contenha apenas os valores ímpares:


                    [1,5,3]

Solução

                    
                        let inteiros = [1, 2, 4, 5, 10, 6, 3];
                        console.log(inteiros);
            
                        let impares = [];
                        inteiros.forEach((item) => {
                            if (item % 2 !== 0) {
                                impares.push(item);
                            }
                        });
                        console.log(impares);
