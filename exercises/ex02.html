Exercício 2

Dada uma variável com um ano, let ano=2004; determinar se o ano é bissexto. Ver imagem anexa.

NÃO PROCURAR A SOLUÇÃO NA INTERNET!!!!!!

Solução

                    
                        let ano = 1900;

                        if (ano % 4 === 0 && ano % 100 !== 0) {
                            console.log(`O ano ${ano} é bissexto.`);
                        } else if (ano % 4 === 0 && ano % 100 === 0 && ano % 400 === 0) {
                            console.log(`O ano ${ano} é bissexto.`);
                        } else {
                            console.log(`O ano ${ano} não é bissexto.`);
                        }
