Exercício 10

Gerar uma chave aleatória do euromilhões (5 números + 2 estrelas) - não podem existir números repetidos!

Os números podem variar entre 1 e 50, e as estrelas entre 1 e 12.

Utilizar o método Math.random().

A chave deve ser apresentada com os números/estrelas ordenados.

Solução

                    
                        let jackpot = [];
            
                        while (jackpot.length < 7) {
                            let newNumber = Math.floor(Math.random() * 51);
                            if (!jackpot.includes(newNumber)) {
                                jackpot.push(newNumber);
                            }
                        }
            
                        let jackpotNumbersOrdered = jackpot
                            .slice(0, 5)
                            .sort(function (a, b) {
                                return a - b;
                            });
            
                        let jackpotStarsOrdered = jackpot.slice(5).sort(function (a, b) {
                            return a - b;
                        });
            
                        let jackpotNumbersText = jackpotNumbersOrdered.join(" ");
                        let jackpotStarsText = jackpotStarsOrdered.join(" ");
            
                        console.log(`${jackpotNumbersText} + ${jackpotStarsText}`);
