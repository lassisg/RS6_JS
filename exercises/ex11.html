Exercício 11

Remova os números repetidos de um array.

Por exemplo, partindo do array,

                    
                        [12, 6, 3, 12, 8, 6, 5]

o resultado deverá ser:

                    
                        [12, 6, 3, 8, 5]

Solução

                    
                        let repeatingValues = [12, 6, 3, 12, 8, 6, 5];
                        console.log(repeatingValues);
            
                        let distinctValues = [];
                        repeatingValues.forEach((value) => {
                            if (!distinctValues.includes(value)) {
                                distinctValues.push(value);
                            }
                        });
                        console.log(distinctValues);
